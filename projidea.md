# Project Vision: FemCare RAG Assistant

## 🌸 Core Purpose
A specialized, safe, and private web application designed to answer women's health questions. This tool aims to give users reliable, clear information about symptoms, wellness, and cycles without the anxiety of scrolling through scary, unverified internet search results.

## 🎯 Core Goals
* **Trustworthy Answers:** The assistant must strictly ground its responses in verified medical documentation. No guessing or making things up.
* **Safety & Escalation:** If a user types in a high-risk medical emergency, the system must recognize it, halt normal chat behavior, and display a high-priority warning directing them to professional care.
* **Modern Layout:** A clean sidebar navigation panel on the left with a dedicated, secure-feeling communication workspace on the right.
